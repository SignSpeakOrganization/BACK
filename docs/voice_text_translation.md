# Cadrage de la traduction bidirectionnelle voix ↔ texte

## Objectif

Permettre la traduction entre la voix et le texte pour faciliter la communication
entre les personnes entendantes et les utilisateurs de la Langue des Signes Française.

---

## Fonctionnement

### Voix → Texte

1. L'utilisateur parle dans le micro
2. L'audio est capturé par l'application
3. Le backend traite l'audio
4. Le texte est généré et affiché

---

### Texte → Voix

1. Un texte est généré ou saisi
2. Le système produit un audio
3. L'utilisateur entend la synthèse vocale
